# Chief of Staff Assessment

A two-part interactive assessment tool that helps leaders determine:
1. Whether they need a Chief of Staff (and if so, fractional or full-time)
2. What archetype they need (Strategic, Operational, External-Facing, or Transformation)

## Quick Start

1. Open Terminal
2. Navigate to this folder: `cd cos-assessment`
3. Install dependencies: `npm install`
4. Start locally: `npm start`
5. Open http://localhost:3000 in your browser

## Customize

Open `src/App.jsx` and search for:
- `YOUR_GOOGLE_APPS_SCRIPT_URL_HERE` — replace with your Google Sheets webhook URL
- `you@yourdomain.com` — replace with your actual email

## Deploy to Vercel

1. Push to GitHub
2. Go to vercel.com and import the repo
3. Click Deploy — done!
